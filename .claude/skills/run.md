# Skill: Run Local Development Server

Start the local development server for PDF Tools.

## Instructions

1. Start the Vite development server:
```bash
cd C:/Users/flavi/source/repos/app1 && npm run dev
```

2. The server will start on **http://localhost:5173/**

3. Run this command in the background so the conversation can continue.

4. After starting, inform the user that the server is running and provide the URL.

## Notes

- The server uses Vite 8 with Hot Module Replacement (HMR)
- Changes to code are reflected immediately
- To stop the server, use Ctrl+C or kill the background process
